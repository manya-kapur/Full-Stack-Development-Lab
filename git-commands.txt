Git Commands Used 

Initializes a new Git repository.
git init

Creates a local copy of a remote repository.
git clone https://github.com/username/repository-name.git

Displays the current state of the working directory.
git status

Stages all modified files for commit.
git add .

Saves changes with a message.
git commit -m "Initial commit"

Uploads local commits to the remote repository.
git push origin main

Fetches and merges changes from remote repository.
git pull origin main

Branching and Merging

Creates and switches to a new branch.
git branch feature-branch
git checkout feature-branch

Merges changes from feature branch into main branch.
git checkout main
git merge feature-branch
